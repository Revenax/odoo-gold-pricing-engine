Gold Pricing Engine

Automated gold pricing with live API updates and POS price enforcement.

  • Automated gold price updates from external API (every 10 minutes)
  • Product pricing based on weight, purity, and markup
  • POS price enforcement to prevent sales below minimum price
  • Real-time cost and sale price calculations

Extends product.template with gold-specific fields, automatic price updates via cron, backend and frontend POS validation, and batch processing for performance.
